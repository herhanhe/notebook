## 1.列表、元组操作

### 列表

* 列表是我们最以后最常用的数据类型之一，通过列表可以对数据实现最方便的存储、修改等操作
* 定义列表
  
  ```python
    names = ['herhan',"Edison",'Sum']
  ```

* 通过下标访问列表中的元素，下标从0开始计数，还可以倒着取

  ```python
    >>> names[0]
    'herhan'
    >>> names[2]
    'Sum'
    >>> names[-1]
    'Sum'
    >>> names[-2]
    'Edison'
  ```

* **切片**：取多个元素

  ```python
    names = ["herhan"]
  ```
